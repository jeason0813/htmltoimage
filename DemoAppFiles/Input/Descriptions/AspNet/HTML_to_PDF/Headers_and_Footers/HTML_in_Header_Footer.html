EVO HTML to PDF Converter offers a great flexibility in adding headers and footers to the generated PDF document. You can add in header and footer any PDF element that you can normally add in a PDF page : HTML, text, image.

The options you can set in this demo are:

HTML Page to Convert
  • HTML Page URL or Local File to Convert. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
Enable Header and Footer
  • Add Header. Using this option you can enable or disable the header in the generated PDF document
  • Add Footer. Using this option you can enable or disable the footer in the generated PDF document
Header Options
  • Show in First Page. Using this option you can enable or disable the header in the first page of the generated PDF document
  • Show in Odd Pages. Using this option you can enable or disable the header in the odd pages of the generated PDF document. If want to disable the header in first page too then you have to explicitly disable the header in first page using the option above
  • Show in Even Pages. Using this option you can enable or disable the header in the even pages of the generated PDF document
  • Draw Header Line. This option controls if a thin line will be drawn at the bottom of the header
  • Spacing. This option represents the space between the header and the PDF page body
  • First Page Spacing. This option represents the space between the header and the PDF page body in the first page of the generated PDF document
Footer Options
  • Show in First Page. Using this option you can enable or disable the footer in the first page of the generated PDF document
  • Show in Odd Pages. Using this option you can enable or disable the footer in the odd pages of the generated PDF document. If want to disable the footer in first page too then you have to explicitly disable the footer in first page using the option above
  • Show in Even Pages. Using this option you can enable or disable the footer in the even pages of the generated PDF document
  • Add Page Numbers. This option controls if the page numbering will be added in footer
  • Draw Footer Line. This option controls if a thin line will be drawn at the top of the footer
  • Spacing. This option represents the space between the footer and the PDF page body
