EVO HTML to PDF Converter allows you to convert another page of the same application in the same ASP.NET session. The values you filled in HTML form will be saved in session object and used in the HTML page converted to PDF.

The options you can set in this demo are:
  • Fill the Session Values to Capture in PDF. These are the HTML form fields where you can enter the values to be stored in ASP.NET session and displayed by the HTML page to be converted to PDF
  • Preview HTML Page to Convert. This is a link to the HTML page to convert to PDF using the values entered in HTML form and stored in ASP.NET session
