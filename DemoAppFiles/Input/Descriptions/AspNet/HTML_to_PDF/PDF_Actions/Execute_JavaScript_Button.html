EVO HTML to PDF Converter allows you to add an Acrobat JavaScript code in generated PDF to be executed when a button of a PDF form is clicked. In this demo you can set button actions in the generated PDF document to display an alert message, automatically open the print dialog or set the zoom level of the generated PDF document.

The options you can set in this demo are:

HTML Page to Convert
  • HTML String with a Button. When you convert a HTML string you also have the option to specify a base URL that will be used by converter to resolve the relative URLs found in the HTML string to fully qualified URLs. If your HTML string uses only fully qualified URLs then this parameter is not necessary
Acrobat JavaScript Actions
  • Display an Alert Message. When this type of action is selected you can also set an alert message to be displayed when the button is clicked
  • Open Print Dialog. When this type of action is selected the print dialog is opened when the button is clicked
  • Set Zoom Level. When this type of action is selected you also set a zoom level to be used by the PDF viewer when the button is clicked
