EVO HTML to PDF Converter can set the location in a PDF page where to go when the generated PDF is opened in a PDF viewer. Additionally you can set the initial zoom level.

The options you can set in this demo are:

HTML Page to Convert
  • HTML Page URL or Local File to Convert. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
Go To Page Action
  • Page Number. The number of the PDF page where to go when the generated PDF document is opened in a PDF viewer
  • Location. The point in the PDF page where to go when the generated PDF document is opened in a PDF viewer
  • View Mode. The view mode when the PDF viewer goes to the selected location in the selected PDF page. When the view mode is X, Y and Zoom you can also set a zoom level to be used by PDF viewer. The other options are Fit Window to fit the PDF viewer window both horizontally and vertically, Fit Horizontally to fit the PDF viewer window horizontally and Fit Vertically to fit the PDF viewer window vertically
