EVO HTML to PDF Converter automatically converts all the internal links from HTML to internal links in PDF. You can disable this behavior completely or you can selectively disable the conversion of certain internal links setting the data-internal-link-enabled attribute value to false.

The following options are available in this demo:

  • HTML Page URL or Local File to Convert. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
  • Enable the Internal Links in PDF. Using this option you can enable or disable the conversion of the internal links from HTML to internal links in PDF
