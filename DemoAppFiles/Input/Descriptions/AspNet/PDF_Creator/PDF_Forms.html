EVO HTML to PDF Converter allows you to create a PDF form with various types of fields in a PDF document. You can save the created PDF document to a file, a stream or in a memory buffer.

The basic options you can set in this demo are grouped in a few categories.

Submit URL
  • PDF Form Submit URL. This option allows you set the URL of the web page where to submit PDF form fields when the Submit button from PDF form is clicked
Submit Method
  • POST. When this option is selected the generated the PDF form fields will be sent to web page using the POST HTTP method
  • GET. When this option is selected the generated the PDF form fields will be sent to web page using the GET HTTP method
