EVO HTML to PDF Converter allows you to add images to a PDF document. You can control the images size and scaling in PDF document. You can save the created PDF document to a file, a stream or in a memory buffer.

In this demo you can see how to draw unscaled images to a PDF document, how to scale images to fit a given box either preserving or not the aspect ratio, how to draw transparent images and rotated images in a PDF document.
