EVO HTML to PDF Converter allows you to create empty PDF documents to which you can add various types of elements: HTML, text, images, graphics. You can save the created PDF document to a file, a stream or in a memory buffer.

The basic options you can set in this demo are grouped in a few categories.

PDF Standard
  • Full PDF. When this option is selected the generated PDF document can use all the features of the PDF standard
  • PDF/A-1b. When this option is selected the generated PDF document will be compliant to PDF/A-1b standard
  • PDF/X-1a. When this option is selected the generated PDF document will be compliant to PDF/X-1a standard
Color Space
  • RGB. When this option is selected the generated PDF document will use the RGB color space
  • Gray Scale. When this option is selected the generated PDF document will use the Gray Scale color space
  • CMYK. When this option is selected the generated PDF document will use the CMYK color space
PDF Page Options
  • PDF Page Size. You can set the page size of the generated PDF document to a standard size like A4, Letter, Legal, etc. or you can set custom PDF page size giving the width and height in points. 1 point is 1/72 inches. By default the PDF page size is A4
  • PDF Page Orientation. You can set the page orientation of the generated PDF document to Portrait or to Landscape. By default the page orientation is Portrait
  • PDF Page Margins. You can individually set the left, right, top and bottom page margins of the generated PDF document. The margins are expressed in points. 1 point is 1/72 inches. By default the PDF page margins are all 0 points
