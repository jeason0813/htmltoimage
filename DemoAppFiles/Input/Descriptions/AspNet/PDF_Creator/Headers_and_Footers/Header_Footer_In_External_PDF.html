EVO HTML to PDF Converter EVO HTML to PDF Converter allows you to apply headers and footers on the external PDF documents added to a PDF document. You can control the visibility of the header and footer in first page of the external PDF document using the AppendDocument() and InsertDocument() methods parameters.

The following options are available in this demo.

  • HTML Page to Convert URL. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
  • PDF File to Insert Before Conversion. This represents the external PDF file to be inserted in the final PDF document before the HTML to PDF conversion result. This feature can be used for example to add a cover page in final PDF document
  • Add Header and Footer in Inserted PDF Document. This option can be used to enable or disable the header and footer in the external PDF document inserted before the HTML to PDF conversion result
  • Show Header in First Page. This option can be used to enable or disable the header in the first page of the external PDF document inserted before the HTML to PDF conversion result
  • Show Footer in First Page. This option can be used to enable or disable the footer in the first page of the external PDF document inserted before the HTML to PDF conversion result
  • PDF File to Add After Conversion. This represents the external PDF file to be added in the final PDF document after the HTML to PDF conversion result
  • Add Header and Footer in Appended PDF Document. This option can be used to enable or disable the header and footer in the external PDF document added after the HTML to PDF conversion result
