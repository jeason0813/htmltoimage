EVO HTML to PDF Converter allows you add HTML in header and footer. The HTML you add in header and footer as HtmlToPdfVariableElement objects can contain the &p; place holder for current PDF page number and the &P; place holder for the total number of pages of the PDF document.

The following options are available in this demo:

  • HTML Page URL or Local File to Convert. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
  • HTML String with Page Numbering to Add in Footer. The HTML string to be rendered in PDF document footer containing the &p; and &P; place holders for current page number and the total number of pages of the PDF document. Optionally you can also set a base URL to be used to resolve the relative resources URLs to full URLs
  • Draw Footer Line. This option controls if a thin line will be drawn at the top of the footer
  • Spacing. This option represents the space between the footer and the PDF page body
