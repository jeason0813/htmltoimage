EVO HTML to PDF Converter allows you to add an Acrobat JavaScript code in generated PDF to be executed when the PDF document is opened in a PDF viewer. In this demo you can set open actions in the generated PDF document to display an alert message, automatically open the print dialog or set the zoom level of the generated PDF document.

The options you can set in this demo are:

HTML Page to Convert
  • HTML Page URL or Local File to Convert. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
Acrobat JavaScript Actions
  • Display an Alert Message. When this type of action is selected you can also set an alert message to be displayed by the PDF viewer when the generated PDF document is opened
  • Open Print Dialog. When this type of action is selected the PDF viewer will automatically open the print dialog when the generated PDF document is opened
  • Set Zoom Level. When this type of action is selected you can also set a zoom level to be used by the PDF viewer when the generated PDF document is opened
