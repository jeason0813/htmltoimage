EVO HTML to PDF Converter allows you to override the default document header or footer with a custom header or footer in one or many pages of the PDF document. The override is made in the handler of the PrepareRenderPdfPageEvent event of the HtmlToPdfConverter class.

The following options are available in this demo:

  • HTML Page to Convert URL. The URL of the HTML page to convert must be accessible from the computer where the converter runs. If the page requires authentication or if you are using a proxy server to access Internet then you have to set the converter properties accordingly as described in documentation
  • Change the Header and Footer in First Page. Using this option you can enable or disable the default header and footer overriding in the first page of the generated PDF document
