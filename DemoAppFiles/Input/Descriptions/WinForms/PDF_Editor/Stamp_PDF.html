EVO HTML to PDF Converter allows you to add watermarks and stamps to a PDF document. You can add any PDF element in a watermark or a stamp repeatead in each PDF page, including HTML. The resulted PDF document can be saved to a file, stream or memory buffer.

The following options are available in this demo:

  • The PDF Document to Stamp. The full path of the PDF document to stamp
  • Stamp Size in PDF Page. You can set the stamp width and height in points, 1 point is 1/72 of an inch
  • Stamp HTML Code. The HTML string to be rendered in the stamp. Optionally you can also set a base URL to be used to resolve the relative resources URLs to full URLs
