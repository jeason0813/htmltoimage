EVO HTML to PDF Converter allows you to split a PDF document into a set of smaller PDF documents. The resulted PDF documents can be saved to files, streams or memory buffers.

The following options are available in this demo:
  • The PDF Document to Split. The full path of the PDF file to split
  • Open the First Resulted PDF. This option is visible if the split succeeded and represents a link to the first PDF file resulted after split
  • Open the Second Resulted PDF. This option is visible if the split succeeded and represents a link to the second PDF file resulted after split
  • Open Initial PDF. This option is visible if the split succeeded and represents a link to the initial PDF document to split
