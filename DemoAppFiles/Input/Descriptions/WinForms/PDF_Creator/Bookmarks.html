EVO HTML to PDF Converter allows you to add a hierarchy of bookmarks to a PDF document. You can save the created PDF document to a file, a stream or in a memory buffer.

In this demo you can see how to create a hierarchy of bookmarks in a PDF document, set the bookmarks text, color and style. You can also see how to set the zoom level and fit options used when the PDF page pointed by a bookmark is displayed in a PDF viewer.
